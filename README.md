# FBLA-PROJECT  Christoper Columbus' Epic Journys!
This project is a fun historical game about Christopher Columbus and his voyages. In the game you sail your ship to different areas of the world to explore them while collecting goods like gold. You find different tribes, some good and some bad. You have to satisfy the King and Queen of Spain by giving them some of your goods and also, survive.

Tables of contents: Voyage 1, Voayge 2, Voyage 3

Usage: You can use the project to learn about Christopher Columbus and his voyages.
